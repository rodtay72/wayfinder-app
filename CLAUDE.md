## Core Canon

Wayfinder is a parent emotional development pathway.

The journal is a mechanism, not the whole product.

Wayfinder is not:
- child diagnosis
- behaviour labelling
- generic Behaviour -> Advice

Wayfinder helps parents recognise where their Cognition, Affect, and Behaviour are not yet aligned with the child's emerging need.

## Official ALIGN

A = Awareness
L = Locate
I = Integrate
G = Growth
N = Navigate

## Core Pathway

Behaviour
-> Need
-> Parent CAB
-> Alignment Check
-> Awareness
-> Growth
-> Navigate / Next Action

## Non-Negotiables

The phrase "My child did something and I don't know why" must never mean:
"The child is the problem."

It means:
"The child's behaviour may be revealing a moment where the parent's CAB is not aligned with the child's emerging need."

Use cautious language:
- may
- might
- possible
- let's explore

Do not diagnose, label, or blame the child or parent.

Do not modify Supabase auth, RLS, SQL, API routes, email verification, parent/child ID generation, journal save/read logic, dashboard loading, privacy masking, or deployment config unless explicitly instructed.
